If a prefix is declared but not used, it is dead code and should be removed.

Examples

The following code snippet would PASS:

prefixes:
    data: <http://data.politie.nl/>

queries:
    DEFINE QUERY myQuery => /data:myItem

The following code snippet would FAIL:

prefixes:
    pol: <http://ontologie.politie.nl/def/>
    data: <http://data.politie.nl/>

queries:
    DEFINE QUERY myQuery => /data:myItem
