An OMT variable must always start with the $-symbol. This includes the variables, parameters and bindings.

Examples

The following code snippet would PASS: 
model:
    MyActivity: !Activity
        variables:
        - $variable
        - $variableWithValue = 'value'

        params:
        - $param (string)

    MyComponent: !Component
        bindings:
            param: $param
The following code snippet would FAIL: 
model:
    MyActivity: !Activity
        variables:
        - variable
