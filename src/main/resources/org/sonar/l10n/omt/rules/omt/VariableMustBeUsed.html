If a local variable is declared but not used, it is dead code and should be removed.
You can declare variables in OMT using

  • variables
  • params
  • bindings

Examples

The following code snippet would PASS:

model:
    MyActivity: !Activity:
        title: $myVariable
        variables:
        - $myVariable

The following code snippet would FAIL:

model:
    MyActivity: !Activity:
        variables:
        - $myVariable
